## Get contact by id

```shell
$ node app.js --action get --id gertd4N3iKsbKpsfyAi65
```

## Add new contact

```shell
$ node app.js --action add --name Leo Style --email leo.style@gmail.com --phone 987-654-3455
```

![Add contact screenshot](https://i.ibb.co/m0GD27t/add.jpg)

